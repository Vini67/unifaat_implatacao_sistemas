- Certificado SSL/TLS

**3. Funcionalidades Dinâmicas:**
- Formulário de contato via Lambda
- Sistema de comentários simples
- Newsletter signup
- Analytics básico

## ⚡ Lambda Function - Formulário de Contato

```python
import json
import boto3
import os
from datetime import datetime

def lambda_handler(event, context):
    # Parse do body
    body = json.loads(event['body']) if event['body'] else {}
    
    # Validação básica
    required_fields = ['name', 'email', 'message']
    for field in required_fields:
        if field not in body or not body[field]:
            return {
                'statusCode': 400,
                'headers': {
                    'Access-Control-Allow-Origin': '*',
                    'Content-Type': 'application/json'
                },
                'body': json.dumps({
                    'error': f'Campo {field} é obrigatório'
                })
            }
    
    # Salvar no DynamoDB
    dynamodb = boto3.resource('dynamodb')
    table = dynamodb.Table(os.environ['CONTACTS_TABLE'])
    
    try:
        table.put_item(
            Item={
                'id': str(datetime.now().timestamp()),
                'name': body['name'],
                'email': body['email'],
                'message': body['message'],
                'timestamp': datetime.now().isoformat(),
                'source': 'blog-contact-form'
            }
        )
        
        # Enviar notificação por email (opcional)
        sns = boto3.client('sns')
        sns.publish(
            TopicArn=os.environ['NOTIFICATION_TOPIC'],
            Subject='Novo contato do blog',
            Message=f"Nome: {body['name']}\nEmail: {body['email']}\nMensagem: {body['message']}"
        )
        
        return {
            'statusCode': 200,
            'headers': {
                'Access-Control-Allow-Origin': '*',
                'Content-Type': 'application/json'
            },
            'body': json.dumps({
                'message': 'Mensagem enviada com sucesso!'
            })
        }
        
    except Exception as e:
        return {
            'statusCode': 500,
            'headers': {
                'Access-Control-Allow-Origin': '*',
                'Content-Type': 'application/json'
            },
            'body': json.dumps({
                'error': 'Erro interno do servidor'
            })
        }
```

## 🚀 Deploy Automatizado

### .github/workflows/deploy.yml
```yaml
name: Deploy Blog to S3

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

jobs:
  deploy:
    runs-on: ubuntu-latest
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Configure AWS credentials
      uses: aws-actions/configure-aws-credentials@v2
      with:
        aws-access-key-id: ${{ secrets.AWS_ACCESS_KEY_ID }}
        aws-secret-access-key: ${{ secrets.AWS_SECRET_ACCESS_KEY }}
        aws-region: us-east-1
    
    - name: Build website
      run: |
        # Minificar CSS e JS
        npm install -g clean-css-cli uglify-js
        cleancss -o website/css/style.min.css website/css/style.css
        uglifyjs website/js/app.js -o website/js/app.min.js
        
        # Atualizar referências
        sed -i 's/style.css/style.min.css/g' website/*.html
        sed -i 's/app.js/app.min.js/g' website/*.html
    
    - name: Deploy to S3
      run: |
        aws s3 sync website/ s3://${{ secrets.S3_BUCKET_NAME }}/ \
          --delete \
          --cache-control "public, max-age=31536000" \
          --exclude "*.html" \
          --exclude "*.xml"
        
        # HTML files with shorter cache
        aws s3 sync website/ s3://${{ secrets.S3_BUCKET_NAME }}/ \
          --delete \
          --cache-control "public, max-age=3600" \
          --include "*.html" \
          --include "*.xml"
    
    - name: Invalidate CloudFront
      run: |
        aws cloudfront create-invalidation \
          --distribution-id ${{ secrets.CLOUDFRONT_DISTRIBUTION_ID }} \
          --paths "/*"
    
    - name: Deploy Lambda Functions
      run: |
        cd lambda/contact-form
        zip -r contact-form.zip .
        aws lambda update-function-code \
          --function-name blog-contact-form \
          --zip-file fileb://contact-form.zip
```
